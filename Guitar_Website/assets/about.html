The Musician's Cheat Sheet

Your resource for finding the key to any guitar song you know!

My name is Nick Pirone and I built this website to help fellow aspiring musicians with their guitar improvisation.

As someone who learned guitar without any music theory background, identifying major keys and figuring out what notes or chords I could play was a frustrating, painstakingly tedious process.

Manually checking notes against a certain key was very repetitive, so I thought "Why not let a computer do it for me?"

I designed the site using Bootstrap and with the idea that it would be Mobile-First--easy to see and use on my IPhone and web browser.

The guitar fretboard is sized based on the user's viewport so that it looks good on any device. The cells in the fretboard are generated programmatically in JQuery--the code for which can be found here!
